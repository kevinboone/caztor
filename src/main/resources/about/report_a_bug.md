# How to report a bug

It's best to report bugs through Caztor's
[GitHub repository](https://github.com//kevinboone/caztor). If practicable, please 
provide enough information to reproduce the bug, including details of the Java
JVM and platform. 

Please include the exact version number of Caztor, from _Help|About_. 

In many cases it will help to run Caztor with maximum logging, and redirect
program output to a file.
